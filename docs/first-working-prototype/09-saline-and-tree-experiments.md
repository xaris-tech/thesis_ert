# Saline and Tree Experiments

## Common Procedure

1. Create experiment record and sample ID.
2. Measure geometry and install E1-E12 clockwise.
3. Photograph E1 orientation and electrode placement.
4. Run hardware status, current, and contact checks.
5. Capture warm-up frames until drift stabilizes.
6. Capture three to five baseline frames.
7. Introduce the controlled anomaly without moving electrodes.
8. Capture the same number of anomaly frames.
9. Reconstruct adjacent and opposite datasets separately.
10. Save ground-truth location, size, depth, and photographs.

## Saline Tank

- Use a circular tank and fixed electrodes every 30 degrees.
- Keep water level, salinity, and temperature stable.
- Start with a nonconductive plastic target.
- Then test a conductive target.
- Test center and edge positions separately.

## Banana Pseudostem

- Use one horizontal plane with consistent insertion depth.
- Allow electrode readings to settle before baseline capture.
- Use a documented cavity or wet/dry insert as the controlled change.
- Expect strong moisture-related changes and contact variability.

## Cut Tree-Trunk Sample

- Record species, diameter, bark condition, and moisture condition.
- Scan intact, then introduce a measured cavity or moisture change.
- Section or inspect the sample afterward when possible.

## Living Tree

- Use only after dummy-load, tank, and sacrificial samples pass.
- Use shallow consistent electrodes and minimize damage.
- Do not create destructive validation defects.
- Treat results as repeatability/feasibility observations unless independent
  ground truth is available.

## Comparison Metrics

- baseline RMS and drift
- forward/reverse disagreement
- current stability by injection pair
- anomaly peak sector versus true sector
- center/edge localization behavior
- repeatability after fixture reinstallation
